# Калина и Карина

Фото галерия. Снимките се четат живо от публична Google Drive папка —
`photos.json` се обновява автоматично на всеки час от GitHub Action
(`.github/workflows/update-photos.yml`). Нови снимки в папката се появяват
сами, без ръчна намеса.

- Хостинг: GitHub Pages
- Домейн: kalinakarina.site
